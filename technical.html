- - - Course Monitoring - - -

While trying to schedule for class last semester I got extremely fustrated and could not get a single class or time I wanted. Every hour I would go check to see if any seats for my classes opened up. But to my disappoint, they never did. So I decided to take matters into my own hands and build my own program that would track courses for me and send me a text message when a seat opened up. I used Python to build the program and Twilio to send the text messages. I used the requests library to make HTTP requests to the course monitoring, and I used the BeautifulSoup library to parse the HTML and find the number of seats available for each class. Now, I have all the classes I want. The best part of this project was recognizes that I could use my skills to solve a problem that I was having. I am very proud of this project and I am excited to see what I can do with it in the future. Click on the icon below to see my github and navigate to my code !!


- - - Howdy Hack 2021 Piano Keyboard - - -

I participated in my first hackathon back in February of 2021. It was a 24 hour hackathon and I was able to build a piano keyboard that could be played with your hands. I used the OpenCV library to detect the hand and the PyGame library to play the notes. I also used the PyAutoGUI library to detect the position of the mouse and the keyboard library to press the keys. I am very proud of this project because it was my first hackathon and I was able to build something that I thought was cool. I am excited to see what I can do with this project in the future. In reality, this project was not very good but we learned alot from this such as how to work in a team and how to use new libraries. I plan on doing more Hackathons but only after I learn to become a better coder. Also, I figured out python is my language of choice.
